# mini-RAG

## About

This repository contains my implementation of the **mini-RAG** project based on the educational course created by **Abu Bakr Soliman**.

I completed this project as part of my Bachelor's thesis at **Centria University of Applied Sciences (Finland)**. The project serves as both a learning journey and the technical foundation for my thesis on Retrieval-Augmented Generation (RAG) systems.

## Acknowledgment

Special thanks to **Abu Bakr Soliman** for creating the mini-RAG educational course that inspired and guided this implementation.

This repository represents my own implementation while following the course, and will later evolve into a production-ready enterprise AI platform in a separate project.
